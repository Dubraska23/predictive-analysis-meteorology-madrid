# predictive-analysis-meteorology-madrid
Modelo de predicción de series temporales para datos meteorológicos de Madrid utilizando la API de Meteostat. Implementa descomposición de tendencia, estacionalidad armónica y modelado ARIMA para la proyección de temperaturas máximas. Enfocado en la trazabilidad del dato y la automatización del flujo de ingesta para Business Analytics (UAM)
